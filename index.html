🧠 What is HTML?

HTML stands for HyperText Markup Language.

  • It is NOT a programming language
  • It is a markup language
  • It tells the browser what content is and how it’s structured

Think of a website like a human body:

  • HTML → skeleton (structure)
  • CSS → clothes (style)
  • JavaScript → brain (behavior)

Right now, we’re building the skeleton.

🧩 What does HTML do?

HTML answers questions like:

  • This is a heading
  • This is a paragraph
  • This is an image
  • This is a link

Browsers (Chrome, Firefox, etc.) read HTML and display it as a webpage.

🏷️ What is an HTML tag?

HTML uses tags.

Example:

<p>Hello World</p>
  • <p> → opening tag
  • </p> → closing tag
  • Hello World → content

This tells the browser:

“This text is a paragraph.”

🧱 Your First HTML Page

Every HTML page has a basic structure.
Here is a complete, correct, real HTML page:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

Don’t memorize yet. Let’s understand it.

🔍 Line-by-Line Explanation

1️⃣ <!DOCTYPE html>

Tells the browser:

“Hey, this is an HTML5 document.”

2️⃣ <html>...</html>

This wraps everything on the page. It’s the root of the document.

3️⃣ <head>...</head>

This is invisible stuff:

  • Page title
  • Meta info
  • CSS links (later)

Users don’t see this directly.

4️⃣ <title>My First Page</title>

This shows:

  • In the browser tab
  • NOT on the page itself

5️⃣ <body>...</body>

Everything inside here shows on the page.

6️⃣ <h1>

A main heading (big and important). There are h1 to h6 (big → small).

7️⃣ <p>

A paragraph of text.

🧪 Try This Yourself (IMPORTANT)

  1. Open Notepad (Windows) or TextEdit (Mac → plain text)
  2. Paste the code above
  3. Save the file as: index.html
  4. Double-click it
  5. It opens in your browser 🎉

You just made a website.

🧠 Key Rules (Remember These)

  • Tags usually come in pairs
  • HTML is case-insensitive, but we write lowercase
  • Indentation (spaces) is for humans, not browsers
