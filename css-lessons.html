CSS — পাঠ ১: CSS কী? (একদম সহজ)

দারুণ 👍 আজ থেকে আমরা CSS একদম সহজভাবে, ধাপে ধাপে — সবকিছু বাংলায় শিখব 😊 চিন্তা কোরো না, ধরে ধরে বুঝাবো।

CSS কী?

CSS এর পূর্ণরূপ Cascading Style Sheets। এটা ওয়েবসাইটকে সুন্দর করে সাজানোর কাজ করে।

সহজ করে বললে:

HTML → কাঠামো (কি জিনিস)
CSS → সাজসজ্জা (দেখতে কেমন)

উদাহরণ:

HTML বলে: “এটা একটা লেখা”
CSS বলে: “লেখাটা লাল করো, বড় করো, মাঝখানে রাখো”

CSS কীভাবে কাজ করে?

CSS ব্রাউজারকে বলে: “এই জিনিসটা এমনভাবে সাজাও”

CSS লেখার নিয়মটা এমন:

h1 {
  color: blue;
}

এটা বাংলায় পড়ি:

  • h1 → সব h1 শিরোনাম
  • { } → স্টাইল শুরু ও শেষ
  • color → কী বদলাবো
  • blue → কী রঙ করবো

➡️ মানে: “সব h1 লেখার রঙ নীল করো”

প্রথম ছোট উদাহরণ (HTML + CSS)

<!DOCTYPE html>
<html>
<head>
  <style>
    p {
      color: red;
    }
  </style>
</head>
<body>

  <p>আমি CSS শিখছি!</p>

</body>
</html>

এখানে কী হবে?

লেখাটা লাল রঙের হবে ❤️

এটাই CSS-এর কাজ।

শুধু ৩টা শব্দ মনে রাখলেই চলবে

১️. Selector → কোন জিনিস সাজাবো (p, h1)
২️. Property → কী বদলাবো (color)
৩️. Value → কীভাবে বদলাবো (red)

এই তিনটাই আপাতত যথেষ্ট 👍

ছোট অনুশীলন (খুব দরকারি)

তুমি চেষ্টা করো:

  • red বদলে blue করো
  • p বদলে h1 লেখো
  • নিজের একটা লেখা লেখো

তাহলে আমরা শিখব:

CSS — পাঠ ২:
রঙ, লেখার সাইজ (font-size) আর লেখা মাঝখানে আনা (text-align) 🎨✍️

CSS — পাঠ ২

রঙ, লেখার সাইজ ও লেখা মাঝখানে আনা 🎨✍️ আজ আমরা মাত্র ৩টা নতুন জিনিস শিখব।

আজকের ৩টি বিষয়

  • লেখার রঙ
  • লেখার সাইজ
  • লেখা মাঝখানে আনা

১️⃣ লেখার রঙ (color)

আমরা আগেই একটু দেখেছি।

p {
  color: green;
}

➡️ সব p লেখার রঙ সবুজ হবে

তুমি ব্যবহার করতে পারো:

  • red
  • blue
  • green
  • black

২️⃣ লেখার সাইজ (font-size)

p {
  font-size: 20px;
}

➡️ লেখার সাইজ ২০ পিক্সেল

সংখ্যা বাড়ালে লেখা বড় হবে
সংখ্যা কমালে লেখা ছোট হবে

উদাহরণ:

h1 {
  font-size: 40px;
}

৩️⃣ লেখা মাঝখানে আনা (text-align)

p {
  text-align: center;
}

অন্য অপশন:

  • left → বামে
  • right → ডানে
  • center → মাঝখানে

সব একসাথে ব্যবহার করি

<!DOCTYPE html>
<html>
<head>
  <style>
    p {
      color: blue;
      font-size: 24px;
      text-align: center;
    }
  </style>
</head>
<body>

  <p>আমি CSS শিখছি 😄</p>

</body>
</html>

এখানে কী হবে?

  • লেখা নীল হবে
  • লেখা বড় হবে
  • লেখা মাঝখানে থাকবে

ছোট অনুশীলন (অবশ্যই করো 👇)

  • color বদলে নিজের পছন্দের রঙ দাও
  • font-size ৩০px করো
  • text-align left করে দেখো

class কী, কেন ব্যবহার করি, আর খুব সহজ উদাহরণ 🎯

CSS — পাঠ ৩

চল এগোই 😊 আজকের পাঠটা খুব গুরুত্বপূর্ণ, কিন্তু আমি একদম সহজ ভাষায় বুঝাবো।

class কী? কেন দরকার?

একটা সমস্যা ভাবো 🤔

ধরো তোমার কাছে অনেকগুলো লেখা আছে, কিন্তু তুমি শুধু কিছু লেখা আলাদা করে সাজাতে চাও।

p লিখলে → সব p একসাথে বদলে যায়
কিন্তু আমরা চাই → নির্দিষ্ট কিছু

এখানেই আসে class 🎯

class কী?

class হলো একটা নাম যেটা দিয়ে আমরা HTML-এর নির্দিষ্ট জিনিসকে ধরতে পারি।

সহজ করে:

class = “এই জিনিসটা আলাদা”

HTML-এ class ব্যবহার

<p class="redText">এই লেখা লাল হবে</p>
<p>এই লেখা স্বাভাবিক থাকবে</p>

এখানে:

  • প্রথম p-এর class আছে → redText
  • দ্বিতীয় p-এর কোনো class নেই

CSS-এ class লিখি কীভাবে?

CSS-এ class লিখতে ডট (.) ব্যবহার হয়।

.redText {
  color: red;
}

➡️ যাদের class নাম redText, তাদের লেখা লাল হবে

সম্পূর্ণ উদাহরণ (খুব সহজ)

<!DOCTYPE html>
<html>
<head>
  <style>
    .redText {
      color: red;
      font-size: 22px;
      text-align: center;
    }
  </style>
</head>
<body>

  <p class="redText">আমি আলাদা স্টাইল!</p>
  <p>আমি সাধারণ লেখা</p>

</body>
</html>

ফলাফল:

  • প্রথম লেখা লাল, বড়, মাঝখানে
  • দ্বিতীয় লেখা বদলাবে না

মনে রাখার ৩টা কথা 🧠

  • class HTML-এ লেখা হয়
  • CSS-এ class ধরতে . লাগে
  • এক class অনেক জায়গায় ব্যবহার করা যায়

ছোট অনুশীলন 👇

  • class নাম বদলে নিজের মতো করো
  • রঙ বদলাও
  • আরেকটা p লেখায় একই class দাও

পরবর্তী পাঠ 🚀

CSS — পাঠ ৪
background color, border, আর box-এর ধারণা (খুব মজা 😄)

CSS — পাঠ ৪

চল চল 😄 আজকের পাঠটা খুব মজার, কারণ আজ থেকে জিনিসগুলো বক্সের মতো দেখতে শুরু করবে।

Background color, Border & Box ধারণা

আগে একটা গোপন কথা 🤫

ওয়েবসাইটে প্রায় সবকিছুই একটা বক্স। লেখা, ছবি, বাটন — সবই বক্সের ভিতরে থাকে।

১️. Background color (পেছনের রঙ)

.box {
  background-color: lightblue;
}

➡️ বক্সের পেছনের রঙ হালকা নীল

২️. Border (চারপাশের লাইন)

.box {
  border: 2px solid black;
}

এখানে:

  • 2px → লাইনের মোটা
  • solid → সোজা লাইন
  • black → রঙ

৩️. Box তৈরি করি (div ব্যবহার করে)

div হলো একটা সাধারণ বক্স।

<div class="box">
  আমি একটা বক্স
</div>

সব একসাথে ব্যবহার করি

<!DOCTYPE html>
<html>
<head>
  <style>
    .box {
      background-color: lightyellow;
      border: 2px solid blue;
      font-size: 20px;
      text-align: center;
    }
  </style>
</head>
<body>

  <div class="box">
    আমি CSS বক্স শিখছি 😄
  </div>

</body>
</html>

এখন তুমি দেখবে:

  • পেছনে রঙ
  • চারপাশে বর্ডার
  • লেখাটা বক্সের ভেতরে

খুব গুরুত্বপূর্ণ নতুন শব্দ 🧠

  • div → একটা বক্স
  • background-color → পেছনের রঙ
  • border → চারপাশের লাইন

ছোট অনুশীলন 🎯

  • background-color বদলাও
  • border-এর রঙ বদলাও
  • আরেকটা div যোগ করো

পরবর্তী পাঠ 🚀

CSS — পাঠ ৫
padding আর margin — বক্সের ভেতর আর বাইরের ফাঁকা জায়গা 📦➡️📦

CSS — পাঠ ৫

আজ আমরা শিখব খুব গুরুত্বপূর্ণ একটা বিষয় 😄 padding আর margin — বক্সের ভেতর আর বাইরের ফাঁকা জায়গা 📦➡️📦

আগে সহজভাবে বুঝি 🤔

একটা বক্স কল্পনা করো।

  • বক্সের ভেতরের ফাঁকা জায়গা = padding
  • বক্সের বাইরের ফাঁকা জায়গা = margin

১️⃣ Padding (ভেতরের ফাঁকা জায়গা)

.box {
  padding: 20px;
}

➡️ লেখার চারপাশে ভিতরে ২০px ফাঁকা জায়গা হবে

Padding দিলে লেখা বর্ডার থেকে একটু দূরে চলে যায়।

২️⃣ Margin (বাইরের ফাঁকা জায়গা)

.box {
  margin: 30px;
}

➡️ বক্সের চারপাশে বাইরে ৩০px ফাঁকা জায়গা হবে

Margin বক্সকে অন্য জিনিস থেকে দূরে সরায়।

সব একসাথে ব্যবহার করি

<!DOCTYPE html>
<html>
<head>
  <style>
    .box {
      background-color: lightblue;
      border: 2px solid black;
      padding: 20px;
      margin: 30px;
      text-align: center;
    }
  </style>
</head>
<body>

  <div class="box">
    আমি padding আর margin শিখছি 😄
  </div>

  <div class="box">
    আমি আরেকটা বক্স
  </div>

</body>
</html>

এখন তুমি দেখবে:

  • লেখা বর্ডার থেকে দূরে (padding)
  • দুই বক্সের মাঝে ফাঁকা জায়গা (margin)

খুব গুরুত্বপূর্ণ মনে রাখো 🧠

  • padding = ভেতরের ফাঁকা
  • margin = বাইরের ফাঁকা
  • দুটোই px দিয়ে মাপা হয়

ছোট অনুশীলন 🎯

  • padding 40px করো
  • margin 10px করো
  • একটা বক্সে আলাদা রঙ দাও

পরবর্তী পাঠ 🚀

CSS — পাঠ ৬
width, height আর বক্সের সাইজ নিয়ন্ত্রণ 📐

CSS — পাঠ ৬

আজ আমরা শিখব বক্সের সাইজ কিভাবে নিয়ন্ত্রণ করতে হয় 📐 width আর height ব্যবহার করে।

কেন দরকার?

অনেক সময় আমরা চাই —

  • বক্স ছোট হোক
  • বক্স বড় হোক
  • সব বক্স সমান সাইজের হোক

তখনই আমরা ব্যবহার করি width আর height।

১️⃣ width (প্রস্থ)

.box {
  width: 300px;
}

➡️ বক্সের প্রস্থ হবে ৩০০px

২️⃣ height (উচ্চতা)

.box {
  height: 150px;
}

➡️ বক্সের উচ্চতা হবে ১৫০px

সব একসাথে ব্যবহার করি

<!DOCTYPE html>
<html>
<head>
  <style>
    .box {
      background-color: lightgreen;
      border: 2px solid black;
      width: 300px;
      height: 150px;
      padding: 20px;
      margin: 20px;
      text-align: center;
    }
  </style>
</head>
<body>

  <div class="box">
    আমি নির্দিষ্ট সাইজের বক্স 😄
  </div>

</body>
</html>

এখন তুমি দেখবে:

  • বক্সের নির্দিষ্ট প্রস্থ
  • বক্সের নির্দিষ্ট উচ্চতা
  • ভেতরে padding আছে

গুরুত্বপূর্ণ কথা 🧠

  • width = বক্স কতটা চওড়া
  • height = বক্স কতটা লম্বা
  • padding দিলে ভেতরের জায়গা বাড়ে
  • margin দিলে বাইরের ফাঁকা বাড়ে

ছোট অনুশীলন 🎯

  • width 200px করো
  • height 200px করো
  • ৩টা বক্স বানাও একই class দিয়ে

পরবর্তী পাঠ 🚀

CSS — পাঠ ৭
display property (block, inline, inline-block) — খুব গুরুত্বপূর্ণ 🔥

CSS — পাঠ ৭

আজ আমরা শিখব display property 😄 block, inline, inline-block — খুবই গুরুত্বপূর্ণ!

display কী?

display ঠিক করে দেয় — একটি HTML element কীভাবে আচরণ করবে।

  • পুরো লাইন নেবে?
  • নাকি শুধু লেখার যতটুকু দরকার ততটুকু?
  • নাকি মাঝামাঝি কিছু?

১️⃣ block

block element পুরো লাইন দখল করে। 

div {
  display: block;
}

block-এর বৈশিষ্ট্য:

  • পুরো width নেয়
  • নতুন লাইনে শুরু হয়
  • width ও height কাজ করে

উদাহরণ: div, p, h1

২️⃣ inline

inline element শুধু লেখার যতটুকু দরকার ততটুকু জায়গা নেয়। 

span {
  display: inline;
}

inline-এর বৈশিষ্ট্য:

  • নতুন লাইনে যায় না
  • width ও height কাজ করে না
  • শুধু content অনুযায়ী সাইজ

উদাহরণ: span, a, strong

৩️⃣ inline-block

inline-block হলো দুইটার মিশ্রণ 😄 

.box {
  display: inline-block;
  width: 150px;
  height: 100px;
}

inline-block-এর বৈশিষ্ট্য:

  • নতুন লাইনে যায় না
  • width ও height কাজ করে
  • পাশাপাশি বসানো যায়

সব একসাথে উদাহরণ

<!DOCTYPE html>
<html>
<head>
  <style>
    .box {
      display: inline-block;

  • প্রতিটির নির্দিষ্ট width ও height আছে

CSS — পাঠ ৮

আজ আমরা শিখব position property 😄 relative আর absolute — layout control শুরু 🔥

position কী?

position ঠিক করে দেয় — একটা element তার জায়গা কিভাবে নেবে।

আমরা আজ শিখব ২টা গুরুত্বপূর্ণ type:

  • relative
  • absolute

১️⃣ position: relative

.box {
  position: relative;
  left: 20px;
}

➡️ element তার নিজের জায়গা থেকে সরে যায়।

  • left → ডানে সরায়
  • right → বামে সরায়
  • top → নিচে সরায়
  • bottom → উপরে সরায়

গুরুত্বপূর্ণ: জায়গাটা আগের মতোই ধরে রাখে।

২️⃣ position: absolute

.box {
  position: absolute;
  top: 0;
  right: 0;
}

➡️ element পুরোপুরি আলাদা হয়ে যায়।

  • সাধারণ flow থেকে বের হয়ে যায়
  • নিকটবর্তী positioned parent-এর উপর ভিত্তি করে বসে

খুব গুরুত্বপূর্ণ বিষয় ⚠️

যদি parent-এ position: relative; থাকে, তাহলে absolute element ওই parent-এর ভিতরে অবস্থান নেবে।

সম্পূর্ণ উদাহরণ

<!DOCTYPE html>
<html>
<head>
  <style>
    .parent {
      position: relative;
      width: 300px;
      height: 200px;
      background-color: lightgray;
    }

    .child {
      position: absolute;
      top: 20px;
      right: 20px;
      background-color: coral;
      padding: 10px;
    }
  </style>
</head>
<body>

  <div class="parent">
    <div class="child">
      আমি কোণায় 😄
    </div>
  </div>

</body>
</html>

এখন তুমি দেখবে:

  • parent একটা বড় বক্স
  • child ডান উপরের কোণায় বসে আছে

মনে রাখো 🧠

  • relative → নিজের জায়গা থেকে সরে
  • absolute → flow থেকে বের হয়ে যায়
  • absolute সাধারণত parent-এর উপর নির্ভর করে

ছোট অনুশীলন 🎯

  • child-কে bottom: 10px করে দেখো
  • left: 50px দিয়ে সরাও
  • parent-এর size বদলাও

পরবর্তী পাঠ 🚀

CSS — পাঠ ৯

CSS — পাঠ ৯

আজ আমরা শিখব Flexbox 😄 আধুনিক layout system — বক্স সাজানো এখন অনেক সহজ 🔥

Flexbox কী?

Flexbox হলো এমন একটা CSS system, যেটা দিয়ে আমরা সহজে element পাশাপাশি, মাঝখানে বা সমান দূরত্বে সাজাতে পারি।

আগে inline-block দিয়ে করতাম 😅 এখন Flexbox দিয়ে অনেক সহজ।

Flexbox ব্যবহার শুরু

Flexbox ব্যবহার করতে হলে parent-এ লিখতে হবে: 

.container {
  display: flex;
}

➡️ এখন container-এর ভিতরের সব element flex item হয়ে যাবে।

justify-content (আড়াআড়ি সাজানো)

.container {
  display: flex;
  justify-content: center;
}

অপশন:

  • flex-start → বামে
  • center → মাঝখানে
  • flex-end → ডানে
  • space-between → সমান দূরত্ব

align-items (উল্লম্বভাবে সাজানো)

.container {
  display: flex;
  align-items: center;
}

➡️ element গুলো উপরে-নিচে মাঝখানে বসবে।

সম্পূর্ণ উদাহরণ

<!DOCTYPE html>
<html>
<head>
  <style>
    .container {
      display: flex;
      justify-content: space-between;
      align-items: center;
      height: 200px;
      background-color: lightgray;
    }

    .box {
      width: 100px;
      height: 100px;
      background-color: coral;
      text-align: center;
      line-height: 100px;
    }
  </style>
</head>
<body>

  <div class="container">
    <div class="box">1</div>
    <div class="box">2</div>
    <div class="box">3</div>
  </div>

</body>
</html>

এখন তুমি দেখবে:

  • তিনটা বক্স পাশাপাশি
  • সমান দূরত্বে
  • উল্লম্বভাবে মাঝখানে

মনে রাখো 🧠

  • display: flex → Flexbox চালু
  • justify-content → আড়াআড়ি নিয়ন্ত্রণ
  • align-items → উপরে-নিচে নিয়ন্ত্রণ

ছোট অনুশীলন 🎯

  • justify-content: center করো
  • align-items: flex-start করে দেখো
  • ৪টা box বানাও

আজ আমরা শিখব Responsive Design 😄 মানে — মোবাইল অনুযায়ী layout বদলানো 📱🔥

Responsive Design কী?

Responsive design মানে — ওয়েবসাইট সব device-এ সুন্দর দেখাবে।

  • মোবাইল 📱
  • ট্যাবলেট 📲
  • ল্যাপটপ 💻

একটাই website, কিন্তু screen অনুযায়ী layout বদলাবে।

Media Query কী?

Media Query দিয়ে আমরা বলি — "যদি screen ছোট হয়, তাহলে এই style ব্যবহার করো।" 

@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

➡️ যদি screen 600px বা তার কম হয়, তাহলে background হালকা নীল হবে।

Responsive Flexbox উদাহরণ

<!DOCTYPE html>
<html>
<head>
  <style>
    .container {
      display: flex;
      gap: 10px;
    }

    .box {
      flex: 1;
      background-color: coral;
      padding: 20px;
      text-align: center;
    }

    /* Mobile version */
    @media (max-width: 600px) {
      .container {
        flex-direction: column;
      }
    }
  </style>
</head>
<body>

  <div class="container">
    <div class="box">Box 1</div>
    <div class="box">Box 2</div>
    <div class="box">Box 3</div>
  </div>

</body>
</html>

কি হবে?

  • বড় screen → বক্সগুলো পাশাপাশি
  • ছোট screen (মোবাইল) → বক্সগুলো একটার নিচে একটা

মনে রাখো 🧠

  • @media দিয়ে responsive করা হয়
  • max-width মানে সর্বোচ্চ screen size
  • Flexbox + Media Query = শক্তিশালী combo 🔥

ছোট অনুশীলন 🎯

  • 600px বদলে 800px করো
  • Mobile-এ background বদলাও
  • ৫টা box বানাও

পাঠ ১৫
Button hover animation & smooth effects ✨🔥

CSS — পাঠ ১৫

আজ আমরা শিখব Button Hover Animation 😄 ওয়েবসাইটে smooth effect দিলে design অনেক সুন্দর লাগে ✨🔥

Hover কী?

যখন মাউস কোনো জিনিসের উপর যায়, তখন যেটা ঘটে সেটাই hover effect।

CSS-এ hover লিখি এভাবে: 

button:hover {
  background: red;
}

➡️ মাউস নিলে button লাল হয়ে যাবে।

Smooth Animation (transition)

শুধু color বদলালে হঠাৎ বদলায় 😅 Smooth করতে আমরা ব্যবহার করি transition 

button {
  transition: 0.3s;
}

➡️ 0.3 সেকেন্ডে ধীরে ধীরে পরিবর্তন হবে।

সম্পূর্ণ উদাহরণ

<!DOCTYPE html>
<html>

  font-family: Arial;

}

  font-size: 16px;
  background: darkblue;
  color: white;
  border: none;
  border-radius: 6px;
  cursor: pointer;
  transition: 0.3s;
}

button:hover {
  background: orange;
  transform: scale(1.1);
}

</style>
</head>

  • Hover করলে রঙ বদলাবে
  • Button একটু বড় হবে
  • সবকিছু smooth হবে

মনে রাখো 🧠

  • :hover → মাউস গেলে effect
  • transition → smooth animation
  • transform: scale() → বড়/ছোট করা

ছোট অনুশীলন 🎯

  • Hover-এ background green করো
  • scale(1.2) করে দেখো
  • border যোগ করো

পাঠ ১৬
CSS Animation (keyframes) — চলমান effect 🔥✨

CSS — পাঠ ১৬

আজ আমরা শিখব CSS Animation (keyframes) 🔥 এখন জিনিসগুলো নিজে নিজে নড়বে 😄✨

Animation কী?

Animation মানে কোনো কিছু সময় ধরে ধীরে ধীরে বদলানো।

CSS-এ আমরা ব্যবহার করি @keyframes

ধাপ ১ — keyframes বানাই

@keyframes moveBox {
  from {
    transform: translateX(0px);
  }
  to {
    transform: translateX(200px);
  }
}

➡️ বক্স ০px থেকে ২০০px ডানে যাবে।

ধাপ ২ — animation ব্যবহার করি

.box {
  animation: moveBox 2s infinite alternate;
}

এখানে:

  • moveBox → animation-এর নাম
  • 2s → ২ সেকেন্ড সময়
  • infinite → বারবার চলবে
  • alternate → সামনে- পেছনে চলবে

সম্পূর্ণ উদাহরণ

<!DOCTYPE html>
<html>
<head>
<style>

body {
  padding: 50px;
}

.box {
  width: 100px;
  height: 100px;
  background: orange;
  animation: moveBox 2s infinite alternate;
}

@keyframes moveBox {
  from {
    transform: translateX(0px);
  }
  to {
    transform: translateX(200px);
  }
}

</style>
</head>

<body>




</body>
</html>

কি হবে?

  • একটা কমলা বক্স দেখা যাবে
  • ডানে যাবে
  • আবার বামে আসবে
  • চলতেই থাকবে 😄

ছোট অনুশীলন 🎯

  • translateY ব্যবহার করে নিচে-উপরে নড়াও
  • background রঙ বদলাও
  • 3s করে ধীরে করো

CSS — পাঠ ২১

দারুণ সিদ্ধান্ত 😎🔥 আগে আমরা CSS strong করব, তারপর JavaScript নেবো 💪 আজ শিখব CSS Position — element কোথায় বসবে সেটা নিয়ন্ত্রণ করা 😄

Position কী?

CSS-এ position দিয়ে আমরা ঠিক করি একটা element স্ক্রিনে কোথায় থাকবে।

১️⃣ static (default)

.box {
  position: static;
}

➡️ এটা default value ➡️ কোনো special control থাকে না

২️⃣ relative

.box {
  position: relative;
  left: 50px;
}

➡️ নিজের জায়গা থেকে ৫০px ডানে যাবে ➡️ কিন্তু আগের জায়গা ধরে রাখবে

৩️⃣ absolute

.box {
  position: absolute;
  top: 50px;
  left: 100px;
}

➡️ পুরো page-এর তুলনায় জায়গা নেয় ➡️ Parent যদি relative হয়, তাহলে তার ভিতরে position নেয়

৪️⃣ fixed

.box {
  position: fixed;
  bottom: 0;
  right: 0;
}

➡️ স্ক্রিনে fixed থাকবে ➡️ Scroll করলেও নড়বে না

পাঠ ২২
Z-index & Layering — কোনটা উপরে থাকবে? 🔥

CSS — পাঠ ২২

আজ আমরা শিখব Z-index & Layering 😎🔥 যখন দুইটা element একটার উপর আরেকটা চলে আসে — তখন কে উপরে থাকবে?

Z-index কী?

z-index দিয়ে আমরা ঠিক করি কোন element সামনে থাকবে আর কোনটা পিছনে থাকবে।

⚠️ মনে রাখবে: z-index কাজ করতে হলে element-এর position থাকতে হবে (relative / absolute / fixed)

সহজ উদাহরণ

.box1 {
  position: absolute;
  background: red;
  z-index: 1;
}

.box2 {
  position: absolute;
  background: blue;
  z-index: 2;
}

➡️ z-index বেশি হলে element উপরে থাকবে ➡️ এখানে নীল box লালের উপরে থাকবে

সম্পূর্ণ উদাহরণ

<!DOCTYPE html>
<html>
<head>
<style>

.box1 {
  position: absolute;
  width: 150px;
  height: 150px;
  background: red;
  top: 50px;
  left: 50px;
  z-index: 1;
}

.box2 {
  position: absolute;
  width: 150px;
  height: 150px;
  background: blue;
  top: 100px;
  left: 100px;
  z-index: 2;
}

</style>
</head>
<body>

<div class="box1"></div>
<div class="box2"></div>

</body>
</html>

কি হবে?

  • দুইটা box একটার উপর আরেকটা আসবে
  • নীল box উপরে থাকবে
  • z-index বদলালে অবস্থান বদলাবে

নতুন শব্দ 🧠

  • z-index → লেয়ার নিয়ন্ত্রণ
  • Layer → উপরে-নিচে স্তর
  • Stacking → একটার উপর আরেকটা বসা
